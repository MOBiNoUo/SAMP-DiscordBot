## 1. Install Modules :
```nodejs
npm i discord.js samp-query
```
## 2. Init Project : 
```nodejs
npm init
```
## 3. Edit config.json 
Set Server IP And BOTTOKEN
## 4. Run Bot : 
```nodejs
node index.js
```
# Commands : 
```nodejs
/help
```
```nodejs
/activity
```
```nodejs
/listplayers
```
